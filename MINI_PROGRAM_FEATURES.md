# 微信小程序功能模块清单 (WeChat Mini Program Features)

本清单旨在为 UI 设计师及 AI 生成工具提供完整的功能概览，用于定制化设计。

## 1. 核心商城模块 (C端用户)

### 1.1 首页 (Home)
*   **沉浸式顶部**：
    *   全屏轮播图 (Banner) + 黑色渐变遮罩。
    *   吸顶式搜索栏 (透明 -> 磨砂白渐变交互)。
    *   消息通知入口 (带未读红点)。
*   **金刚区导航 (Grid Nav)**：
    *   4-5个核心入口 (热门、新品、特惠等)，采用柔和粉彩色系背景。
*   **瀑布流商品列表**：
    *   双列布局，展示商品图、名称、价格、销量、购物车快捷按钮。

### 1.2 分类页 (Category)
*   **场景化导航**：
    *   顶部横向滚动卡片 (取代传统侧边栏)，按生活场景或人群分类。
*   **筛选排序**：
    *   综合 | 销量 | 价格 (升/降序)。
*   **商品列表**：
    *   支持单列大图/双列网格切换。

### 1.3 商品详情页 (Product Detail)
*   **轮播画廊**：支持点击放大预览。
*   **核心信息区**：
    *   超大号价格显示 + 会员/代理价标签。
    *   商品标题 + 副标题 (卖点)。
*   **服务保障**：图标+文字 (正品保证、极速发货等)。
*   **富文本详情**：支持图文混排。
*   **规格选择弹窗**：底部半屏弹窗 (SKU选择)。
*   **底部操作栏**：
    *   客服、购物车图标。
    *   加入购物车、立即购买按钮 (支持代理商采购模式)。

### 1.4 购物车 (Cart)
*   **列表管理**：勾选商品、数量增减、左滑删除。
*   **底部结算栏**：全选、合计金额、去结算按钮。

### 1.5 结算流程 (Checkout)
*   **确认订单页**：
    *   地址选择/新增。
    *   商品清单确认。
    *   备注输入。
    *   支付方式选择 (微信支付)。
*   **支付成功页**：
    *   SweetAlert 风格成功动画。
    *   查看订单/返回首页入口。

### 1.6 个人中心 (User Profile)
*   **高端头部卡片**：
    *   深色背景 + 金色边框头像。
    *   用户昵称 + 等级勋章 (普通/会员/代理/经销商)。
*   **资产悬浮卡片**：
    *   展示累计佣金、可提现余额、团队人数。
*   **订单中心入口**：
    *   状态栏：待付款、待发货、待收货、退款/售后。
*   **常用功能列表**：
    *   地址管理、邀请码分享、关于我们、设置。

---

## 2. 代理商分销模块 (B端功能)

### 2.1 代理商工作台 (Agent Workbench)
*   **入口**：位于个人中心显眼位置 (仅代理商可见)。
*   **核心数据**：
    *   今日收益、本月收益、累计收益。
    *   团队新增人数。
*   **功能入口**：
    *   **推广海报**：生成带专属二维码的商品/店铺海报。
    *   **团队管理**：查看下级代理列表及贡献。
    *   **佣金明细**：查看每一笔收入来源。
    *   **提现申请**：提现到微信零钱/银行卡。

### 2.2 进货与云仓 (Stock System)
*   **云仓补货**：
    *   代理商以批发价采购商品存入云库存。
*   **库存管理**：
    *   查看当前库存数量。
    *   提货/发货记录。
*   **采购入仓流程**：
    *   商品详情页 -> 采购入仓 -> 支付 -> 增加云库存。

---

## 3. 售后与服务 (Service)
*   **订单列表**：
    *   全部、待付款、待发货、待收货、已完成。
    *   操作：去支付、确认收货、申请退款、查看物流。
*   **退款/售后**：
    *   申请退款 (仅退款/退货退款)。
    *   填写退款理由、上传凭证图片。
    *   查看退款进度 (商家审核 -> 填写物流 -> 退款成功)。
*   **地址管理**：
    *   新增、编辑、删除收货地址。
    *   设为默认地址。

---

## 4. 其他功能
*   **搜索功能**：支持关键词搜索商品，带历史记录。
*   **消息通知**：系统公告、订单状态变更通知。
*   **偏好设置**：
    *   个人信息修改 (头像/昵称)。
    *   账号绑定。
