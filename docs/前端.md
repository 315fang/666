# 前端子项目（管理后台 · 代理门户）

---

## 1. 管理后台 `admin-ui/`

- **技术栈**：Vue 3、Vite、Element Plus、Pinia、Axios。  
- **本地**：

```bash
cd admin-ui
npm install
npm run dev
# 一般为 http://localhost:5173/admin/
```

- **接口**：axios `baseURL` 为 **`/admin/api`**；Vite 将 `/admin/api` 代理到后端（默认 `http://127.0.0.1:3001`），见 `vite.config.js`。  
- **生产**：后端 `app.js` 托管 `admin-ui/dist` 到 `/admin`，API 仍为 `/admin/api`。  
- **权限**：路由 `meta.permission` 须与 `backend/config/adminPermissionCatalog.js` 及 `store/user.js` 一致。  
- **复杂字段说明**（如商品独立佣金）：页面 `el-alert` + 详见 [业务规则.md](./业务规则.md)。

## 2. 代理门户 `agent-portal/`

- **技术栈**：Nuxt 3（`ssr: false`）、Tailwind、Pinia；构建多为静态（如 `npm run generate`）。  
- **本地**：

```bash
cd agent-portal
npm install
cp .env.example .env
# NUXT_PUBLIC_API_BASE=http://127.0.0.1:3001  （站点根，不要多写 /api）
npm run dev
# 端口常见 3002，见 nuxt.config.ts
```

- **认证**：与小程序共用 `User` 表；门户 JWT `type: 'portal'`。  
- **请求路径**：代码里写完整 **`/api/portal/...`**（因 `apiBase` 是域名根）。  
- **生产**：`NUXT_PUBLIC_API_BASE` 在构建时注入，换 API 域名需**重新构建**。推荐「门户静态托管 + API 独立域名」；`vercel.json` 可将 `/api/*` 反代到后端，同时后端 `CORS_ORIGINS` 需含门户域名。

### 目录结构（摘要）

- `pages/index.vue`、`join.vue`、`login.vue` — 公开页  
- `pages/dashboard/*` — 工作台、佣金、提现、团队、订单、库存  
- `composables/useApi.ts` — 自动带 `Authorization` 的 `$fetch` 封装  
- `middleware/auth.ts` — 未登录跳转 `/login`

### 常用门户接口

| 功能 | 方法 / 路径 |
|------|-------------|
| 登录 | `POST /api/portal/auth/login` |
| 资料 | `GET /api/portal/auth/profile` |
| 改初始密码 | `POST /api/portal/auth/change-initial-password` |
| 钱包 | `GET /api/portal/wallet/info` |
| 佣金流水 | `GET /api/portal/wallet/commissions` |
| 提现 | `POST /api/portal/wallet/withdraw` |
| 提现记录 | `GET /api/portal/wallet/withdrawals` |
| 工作台 | `GET /api/portal/agent/workbench` |
| 代理订单 | `GET /api/portal/agent/orders` |
| 发货 | `POST /api/portal/agent/ship/:id` |
| 补货 | `POST /api/portal/agent/restock` |
| 货款流水 | `GET /api/portal/agent/stock-logs` |
| 团队 | `GET /api/portal/distribution/team` |
| 统计 | `GET /api/portal/distribution/stats` |
| 佣金统计 | `GET /api/portal/commissions/my-stats` |
| 代理确认订单 | `POST /api/portal/orders/:id/agent-confirm` |

主题变量见 `assets/css/main.css`、`nuxt.config.ts`。
