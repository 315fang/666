# 历史资料归档

这里存放历史阶段资料，不作为当前实现的主入口。

## 分类

- `legacy-docs/`: 已过期但有保留价值的历史说明、总结、修复记录
- `old-plans/`: 已结束或失效的阶段计划
- `design-drafts/`: 设计草稿、截图、原型资料
- `snapshots/`: 对照快照、阶段性留档

## 使用原则

1. 归档资料默认不作为当前实现依据。
2. 当前真相应优先参考根 `README.md`、`docs/README.md`、`docs/audit/`、`docs/plans/`。
3. 若归档资料仍需复用，应先提炼并迁回正式文档，再使用。
