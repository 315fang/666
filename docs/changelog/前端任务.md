## 三、首页板块配置

### 3.1 自定义导航栏

**已启用**：`pages/index/index.json` 中已有 `"navigationStyle": "custom"`，无需修改。

**滚动切换逻辑**（`index.js` `onPageScroll`）：

```js
// 海报高度约 620rpx ≈ 310px（375px 设备），阈值取 280
onPageScroll({ scrollTop }) {
  const should = scrollTop > 280;
  if (should !== this.data.navScrolled) {
    this.setData({ navScrolled: should });
  }
}
```

**两种状态：**

| 状态 | 背景 | 品牌名 | 右侧 |
|------|------|--------|------|
| 初始（顶部） | `transparent` | 白色，居中，serif 字体 | 搜索图标 |
| 滚动后 | `#1C1917` | 白色，左对齐 | 气泡通知浮入 |

> 气泡通知（`currentBubble`）在非滚动态固定于屏幕左侧；
> 滚动后隐藏原气泡，在导航栏右侧显示相同内容。

### 3.2 主海报区（全屏 Swiper）

**后端配置字段**（`/homepage-config` 接口返回的 `configs` 对象内）：

```json
{
  "home_posters": [
    "https://cdn.example.com/poster-1.jpg",
    "https://cdn.example.com/poster-2.jpg"
  ],
  "home_poster_url": "https://cdn.example.com/poster-1.jpg"
}
```

**降级逻辑（前端自动处理）：**
1. 优先使用 `home_posters` 数组（多图轮播）
2. 若为空，则使用 `home_poster_url` 单图包装成数组
3. 若都没有，显示内置渐变 Fallback

**样式参数：**

```
高度：620rpx
autoplay 间隔：4000ms
transition 时长：500ms
指示点默认色：rgba(255,255,255,0.4)
指示点激活色：#D4A84B
图片 mode：aspectFill
```

**图片规格建议：**
- 尺寸：750 × 1000px（2:2.67 比例）
- 格式：JPG，质量 80%，文件 ≤ 300KB

### 3.3 会员专享板块

**位置：** 会员等级卡 + 积分卡之后，新品推荐之前。

**后端配置字段：**

```json
{
  "member_posters": [
    "https://cdn.example.com/member-offer-1.jpg",
    "https://cdn.example.com/member-offer-2.jpg"
  ]
}
```

**未配置时的静态兜底（`getMockMemberPosters`）：**

```js
// 返回本地占位图，上线前替换为真实 CDN 图片
['/assets/images/default-poster.svg', '/assets/images/default-poster.svg']
```

**样式参数：**

```
轮播高度：340rpx
左右边距：24rpx
圆角：16rpx
autoplay 间隔：3500ms
指示点激活色：#D4A84B
```

**板块标题：**
- 主标题：`会员专享`（字号 34rpx，字重 700，颜色 `#1C1917`）
- 副标题：`MEMBER EXCLUSIVES`（字号 20rpx，颜色 `#A8A29E`，letter-spacing 3rpx）

**图片规格建议：**
- 尺寸：702 × 340px（扣除左右 24rpx 边距后的实际展示尺寸）
- 内容建议：限时特惠、新会员权益、积分兑换活动
