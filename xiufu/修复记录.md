### 3. ✅ 修复规格选择数据结构问题
**问题描述**: `pages/product/detail.wxml` 第125行判断 `product.specs`，但后端返回的数据结构中实际使用的是 `product.skus`，没有 `specs` 字段

**影响**: 规格选择区域永远不会显示，用户无法选择商品规格

**修复位置**:
- 文件: `qianduan/pages/product/detail.js`
- 方法: `loadProduct`

**修复内容**:
在 `loadProduct` 方法中，将 `skus` 数据转换为 `specs` 格式:

```javascript
// 将 skus 转换为 specs 格式用于页面展示
let specs = [];
if (product.skus && product.skus.length > 0) {
    // 提取所有规格名称和值
    const specMap = {};
    product.skus.forEach(sku => {
        if (sku.spec_name && sku.spec_value) {
            if (!specMap[sku.spec_name]) {
                specMap[sku.spec_name] = new Set();
            }
            specMap[sku.spec_name].add(sku.spec_value);
        }
    });
    
    // 转换为数组格式
    specs = Object.keys(specMap).map(name => ({
        name: name,
        values: Array.from(specMap[name])
    }));
}

// 将 specs 添加到 product 对象
product.specs = specs;
```

**修复位置**:

- 行号: 第77行

**修复内容**:
将佣金条的点击事件改为使用 button 的 `open-type="share"`:

```xml
<!-- 修改前 -->
<view class="commission-bar" wx:if="{{isAgent && commission > 0}}" bindtap="onShareAppMessage">

<!-- 修改后 -->
```

**问题描述**: `components/product-card/product-card.wxml` 中使用了 `product.image` 和 `product.originalPrice`，但后端实际返回的是 `images` 数组和 `market_price`

- 行号: 第4行和第16行

**修复内容**:
```xml
<!-- 修改前 -->
<image class="image" src="{{product.image || '/assets/images/placeholder.svg'}}" mode="aspectFill"></image>
<text class="original-price" wx:if="{{product.originalPrice}}">¥{{product.originalPrice}}</text>

<!-- 修改后 -->

<text class="original-price" wx:if="{{product.market_price || product.originalPrice}}">¥{{product.market_price || product.originalPrice}}</text>
```

### 6. ✅ 修复星级评分 key 问题
**问题描述**: `pages/product/detail.wxml` 第175行使用 `wx:key="*this"`，当数组中有重复值时会报错

**影响**: 评价区域的星级显示可能出现异常

**修复位置**:
- 文件: `qianduan/pages/product/detail.wxml`
- 行号: 第175行

**修复内容**:
```xml
<!-- 修改前 -->
<image wx:for="{{[1,2,3,4,5]}}" wx:for-item="star" wx:key="*this" 
       class="star-icon {{star <= item.rating ? 'filled' : ''}}" 
       src="/assets/icons/star.svg" mode="aspectFit"/>

<!-- 修改后 -->
<image wx:for="{{[1,2,3,4,5]}}" wx:for-item="star" wx:key="star" 
       class="star-icon {{star <= item.rating ? 'filled' : ''}}" 
       src="/assets/icons/star.svg" mode="aspectFit"/>
```
