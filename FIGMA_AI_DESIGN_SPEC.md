# 臻选商城 (Premium Selection) - 全栈设计与组件开发规范

**文档版本**: 1.0.0
**适用对象**: UI设计师 / Figma AI / 前端开发工程师
**生成目标**: 极简高端、轻奢质感的电商小程序 UI 及配套后台管理系统组件。

---

## 1. 全局设计系统 (Global Design System)

### 1.1 色彩体系 (Color Palette)
基于代码 `app.wxss` 中的 CSS 变量定义。

*   **主色调 (Brand Colors)**:
    *   `Primary`: `#2563EB` (亮蓝) - 用于主按钮、选中状态、关键链接。
    *   `Primary Light`: `#3B82F6` - 悬停/点击态。
    *   `Primary Dark`: `#1D4ED8` - 深色强调。
    *   `Gradient`: Linear Gradient (`#2563EB` -> `#3B82F6`) - 用于图标背景、渐变按钮。

*   **辅助色 (Accent Colors)**:
    *   `Gold/Amber`: `#D97706` / `#F59E0B` - 用于价格、会员等级、促销标签、积分。
    *   `Red`: `#EF4444` - 用于通知红点、错误提示、删除按钮、价格强调。
    *   `Success`: `#10B981` - 用于支付成功、状态完成。

*   **中性色 (Neutrals)**:
    *   `Background`: `#F8FAFC` (冷灰白) - 页面底色。
    *   `Card Background`: `#FFFFFF` (纯白) - 卡片背景。
    *   `Title Text`: `#0F172A` (深蓝黑) - 一级标题、重要数字。
    *   `Body Text`: `#334155` - 正文内容。
    *   `Secondary Text`: `#64748B` - 次要信息、说明文字。
    *   `Placeholder`: `#94A3B8` - 输入框占位符、失效状态。
    *   `Border`: `#E2E8F0` - 边框、分割线。

*   **深色模式 (Dark Mode)**:
    *   `Header Background`: `#0F172A` (午夜蓝) - 用户中心头部、导航栏背景。

### 1.2 排版 (Typography)
*   **Font Family**: `DIN Pro` (用于价格/数字), `-apple-system`, `PingFang SC`.
*   **Scale**:
    *   `H1 / Price-Large`: 40rpx - 56rpx (根据场景).
    *   `H2 / Title`: 32rpx (粗体 600).
    *   `Body`: 28rpx.
    *   `Caption / Tag`: 22rpx - 24rpx.

### 1.3 形状与间距 (Shapes & Spacing)
*   **Border Radius**:
    *   `Card`: 24rpx (大卡片).
    *   `Button`: 999rpx (胶囊型).
    *   `Image`: 16rpx.
    *   `Tag`: 8rpx.
*   **Shadows**:
    *   `Card Shadow`: `0 4rpx 16rpx rgba(15, 23, 42, 0.06)` - 轻微浮起感。

## 2. 微信小程序前端页面详解 (Mini Program Pages)

### 2.1 底部导航栏 (Tabbar)
*   **样式**: 白色背景，上边框细微。
*   **Items**:
    1.  **首页**: Icon (Home), 选中高亮蓝。
    2.  **分类**: Icon (Category), 选中高亮蓝。
    3.  **购物车**: Icon (Cart), 选中高亮蓝。
    4.  **我的**: Icon (User), 选中高亮蓝。

### 2.2 首页 (Home) - `pages/index/index`
*   **顶部导航 (Header)**:
    *   **状态**: 随滚动渐变（透明 -> 磨砂白）。
    *   **搜索条**: 胶囊型，磨砂半透明背景，内含“🔍 搜索商品”文本。
    *   **消息入口**: 圆形铃铛图标，右上角带红点。
*   **Hero 区域**:
    *   **Banner**: 全屏宽轮播图，底部及顶部覆盖黑色渐变遮罩(Overlay)，保证文字清晰度。
*   **金刚区 (Grid Nav)**:
    *   4个固定入口，图标背景色分别为淡蓝、淡粉、淡黄等柔和色系。
    *   图标推荐使用 3D 或多彩扁平风格。
*   **分类条 (Category Bar)**:
    *   横向滚动，胶囊状 Tab，选中态为深色背景白字，未选中为灰底黑字。
*   **商品流 (Product Waterfall)**:
    *   双列瀑布流卡片。
    *   **卡片内容**: 正方形主图 + 左上角 NEW 标签 + 标题(2行) + 标签行 + 价格行(大号数字) + 原价(删除线) + 右下角“选购”胶囊按钮。

### 2.3 分类页 (Category) - `pages/category/category`
*   **布局**: 垂直布局，非传统左侧联动。顶栏固定搜索。
*   **场景导航 (Scenario Nav)**:
    *   顶部横向滚动区。方形卡片 (`140rpx`)，选中时有蓝色边框和浅蓝背景。
    *   内容: 图标 + 文字 (如“居家”、“美妆”)。
*   **筛选栏 (Filter Bar)**:
    *   综合 | 销量 | 价格 (带上下箭头)。悬浮吸顶。
*   **商品列表**:
    *   单列列表模式 (List View)。
    *   左侧方图 (`180rpx`)，右上角“热”标签。
    *   右侧信息: 标题、价格(大)、销量(小灰字)。
    *   **交互**: 右下角圆形深色“+”按钮，点击触发快速加购动画。

### 2.4 商品详情页 (Product Detail) - `pages/product/detail`
*   **首屏**:
    *   方形轮播图，右下角数字页码指示器。
    *   价格栏: 显示会员价/代理价标签 (不同身份看到不同价格)。
    *   **代理商分享条**: (仅代理可见) 金色渐变背景卡片，显示“分享赚 ¥XX”，右侧“立即分享”按钮。
*   **服务保障**:
    *   横排图标列表 (正品、发货、退换)，带箭头可点击查看详情。
*   **规格选择条**:
    *   显示“已选: [规格值]”，点击底部弹窗。
*   **商品详情**:
    *   富文本区域，图片宽度自适应 100%。
*   **底部操作栏 (Action Bar)**:
    *   左侧: 购物车图标(带Badge)。
    *   右侧按钮组(Flex):
        *   (代理商): “📦 采购入仓” (特殊色按钮)。
        *   (通用): “加入购物车” (次级按钮) + “立即购买” (主按钮)。
*   **SKU 弹窗**:
    *   半屏模态框。顶部商品缩略图+价格+库存。
    *   规格组: 圆角标签，选中变色。
    *   数量步进器: `[-] [数字] [+]`。

### 2.5 购物车 (Cart) - `pages/cart/cart`
*   **空状态**: 居中图标 + “去逛逛”按钮。
*   **列表项**:
    *   左侧勾选圈 (Checkbox)。
    *   商品图 + 标题 + 规格(灰色小字)。
    *   价格 + 数量步进器 (右对齐)。
    *   左滑/长按删除功能 (UI上需预留删除图标位置)。
*   **底部结算栏**:
    *   全选 Checkbox。
    *   合计金额 (大号字体)。
    *   “去结算”按钮 (主色背景，如未选则置灰)。

### 2.6 结算确认页 (Checkout) - `pages/order/confirm`
*   **地址卡片**:
    *   有地址: 显示姓名、电话、详细地址、默认标签、右箭头。
    *   无地址: 显示“+ 添加收货地址”。
*   **商品清单**: 简易版列表，仅展示图、名、规格、价格、数量。
*   **备注栏**: 单行输入框。
*   **地步栏**: 左侧“共X件，合计¥XX”，右侧“提交订单”按钮。
*   **成功弹窗**: 居中大 Icon (🎉)，包含“查看订单”和“返回首页”按钮。

### 2.7 个人中心 (User Profile) - `pages/user/user`
*   **头部卡片 (Header)**:
    *   背景: 深蓝黑 (`#0F172A`)。
    *   内容: 头像(带光环)、昵称(带编辑笔Icon)、身份标签(会员/团长/代理)。
    *   **数据资产栏** (悬浮于Header底部): 累计佣金 | 余额 | 团队人数。
*   **代理商入口** (仅代理可见):
    *   通栏 Banner 卡片，带背景纹理，文案“代理商工作台”，右侧“进入 >”。
*   **邀请与库存**:
    *   邀请码显示 + “复制”按钮 + “分享”按钮。
    *   (代理) 显示“云仓库存: XX件”。
*   **订单区域**:
    *   标题“我的订单” + “全部 >”。
    *   图标行: 待付款、待发货、待收货、已完成、售后。
*   **工具与菜单**:
    *   九宫格入口: 分佣中心、团队、钱包、邀请。
    *   列表菜单: 地址、通知、客服、关于。

### 2.8 订单列表与详情 (Order)
*   **列表页**: 顶部 Tabs (全部/待付款/...)。卡片式设计，底部右侧操作按钮组(去支付、确认收货等)。
*   **详情页**:
    *   **状态头**: 对应状态的背景色面板 + 状态描述。
    *   **进度条**: 垂直或水平步骤条 (付款 -> 确认 -> 发货 -> 收货)。
    *   **物流栏**: 显示最新一条物流 + “复制单号”。

### 2.9 分销/代理商中心 (Distribution)
*   **分佣中心 (`pages/distribution/center`)**:
    *   仪表盘: 大号累计佣金，下方余额、待结算。
    *   提现按钮: 醒目。
    *   功能入口: 团队、分销订单、退货管理。
    *   **代理商专区**: 显示库存大数字，待发货数，本月利润。
    *   **图表区**: 最新通知列表。
    *   **Tab列表**: 佣金明细(收入) / 提现记录(支出)。
*   **发货工作台 (`pages/distribution/workbench`)** (仅代理):
    *   库存概览卡片。
    *   Tabs: 待发货 / 待确认 / 已发货。
    *   订单卡片: 显示买家信息，操作按钮“🚚 填单号发货” (弹窗输入物流公司单号)。
*   **采购入仓 (`pages/distribution/restock`)**:
    *   商品选择列表 (带 Checkbox)。
    *   底部数量选择器 (预设 10/50/100/200 按钮)。
    *   底部实时计算总价。
*   **我的团队 (`pages/distribution/team`)**:
    *   三栏数据: 直推 / 二代 / 总数。
    *   成员列表: 头像、昵称、等级、加入时间、业绩贡献。

### 2.10 其他页面
*   **收货地址**: 列表卡片，支持设为默认、编辑、删除。新增/编辑表单页。
*   **搜索页**: 历史搜索(标签式)、热门搜索(带排名数字)、搜索结果列表(同分类页)。
*   **我的钱包**: 余额大字，明细列表(级差利润/佣金/提现)，提现弹窗。
*   **售后申请**: 选择类型(仅退款/退货退款)、原因Picker、金额Input、凭证上传。

## 3. 后台管理系统组件需求 (Backend Admin Dashboard)

### 3.1 布局组件 (Layout)
*   **Sidebar (侧边栏)**: 深色背景，一级菜单(Icon+Text)，二级折叠菜单。
*   **Navbar (顶栏)**: 面包屑导航，右侧管理员头像及下拉菜单。

### 3.2 数据展示组件 (Data Display)
*   **Stat Card (统计卡片)**: 标题 + 主数字 + 环比/同比趋势(红绿箭头) + 迷你图表(Sparkline)。
*   **Data Table (高级表格)**:
    *   顶部: 搜索栏 + 筛选下拉框 + “导出/新增”按钮。
    *   列头: 支持排序。
    *   行操作: 编辑、删除、查看详情 (悬浮或行内按钮)。
    *   分页器: 页码、每页显示条数。
    *   支持多选批量操作。

### 3.3 录入组件 (Data Entry)
*   **Rich Text Editor (富文本编辑器)**: 用于商品详情编辑。
*   **Image Uploader (图片上传)**:
    *   支持拖拽。
    *   支持多图排序 (Grid 布局，可拖动调整顺序)。
    *   上传进度条。
*   **Dynamic SKU Form (动态规格表单)**:
    *   添加规格名 -> 添加规格值 -> 自动生成 笛卡尔积 表格(价格/库存/SKU编码)。

### 3.4 业务组件 (Business Components)
*   **Order Manager (订单管理)**:
    *   多Tab视图 (全部/待发货/退款中)。
    *   订单详情抽屉 (Drawer): 右侧滑出，显示时间轴、商品、收货人、费用信息。
    *   发货弹窗: 选择物流公司、输入单号。
*   **Distribution Tree (分销关系树)**:
    *   可视化展示用户层级关系 (树状图或缩进列表)。
    *   展示每个节点的业绩、下级人数。
*   **Finance Logs (财务流水)**:
    *   资金变动记录，支持按类型(佣金/提现/返利)筛选。
    *   提现审核面板: 列表展示申请，操作“通过”或“拒绝(输入理由)”。

## 4. 交互与动效规范 (Interactions)

1.  **点击反馈**: 所有按钮点击时应有透明度变化 (`opacity: 0.8`) 或 缩放 (`scale: 0.98`)。
2.  **加载状态**:
    *   **页面加载**: 中心 Loading Spinner。
    *   **下拉刷新**: 顶部出现 Loading Dots。
    *   **上拉加载**: 底部显示“加载中...”或“没有更多了”。
    *   **图片加载**: 灰色占位图 -> 图片淡入 (Fade-in)。
3.  **空状态**: 当列表为空时，必须显示占位插画 (Illustration) 和引导按钮。
4.  **弹窗动画**:
    *   **模态框 (Modal)**: 缩放淡入 (Zoom In + Fade In)。
    *   **底部弹窗 (Bottom Sheet)**: 底部滑入 (Slide Up)。

## 5. 文件输出要求 (Deliverables)

Figma AI 应基于上述规范生成：
1.  **Mobile UI Kit**: 包含所有小程序页面的高保真设计稿。
2.  **Admin UI Kit**: 包含后台管理系统的核心页面（仪表盘、商品发布、订单列表、分销管理）。
3.  **Component Set**: 提取出的可复用组件（按钮、卡片、表单元素、图标集）。
