# 文档归档说明

本目录用于存放 `cloud-mp` 已经不再作为当前实现依据的历史文档、阶段总结、交付产物和执行记录。

使用原则：

1. 当前项目真相源优先看根 `README.md`、`docs/CLOUD_MP_DEVELOPMENT_GUIDE.md`、`docs/CLOUDBASE_RELEASE_RUNBOOK.md`、`升级.md`。
2. 只有在追历史决策、核对阶段结果或复盘旧交付时，才进入 `docs/archive/`。
3. 已归档资料默认不再作为当前实现和运行规则的直接依据。

当前归档层级：

- `root-history/`：从仓库根目录迁移下来的历史阶段文档、代码审查记录、执行摘要等资料。
- `root-artifacts/`：从仓库根目录迁移下来的临时图片、调试输出、一次性 JSON 和局部构建碎片。
