# 臻选商城 (Premium Selection) UI 设计规范文档

## 1. 项目概述
本项目是一个 S2B2C 模式的电商平台，包含 **微信小程序端 (C端/小B端)** 和 **后台管理系统 (B端)**。
本此文档旨在为 UI 设计师或 AI 生成工具提供详尽的视觉描述，以确保设计还原度。

---

## 2. 微信小程序端 (Mini Program)

### 2.1 全局设计风格 (Global Style)
*   **核心关键词**：高端、极简、信赖感、现代电商。
*   **色彩体系**：
    *   **主色调**：深邃蓝/黑 (`#0F172A` - `#1E293B`) —— 用于头部背景、重要强调。
    *   **强调色**：琥珀金 (`#D97706` / `#F59E0B`) —— 用于价格、会员标识、促销按钮。
    *   **背景色**：冷灰白 (`#F8FAFC` / `#FFFFFF`) —— 保持页面干净通透。
    *   **辅助色**：中国红 (`#EF4444`) —— 用于通知气泡、错误提示。
*   **字体风格**：无衬线字体，强调数字显示的清晰度（如价格、销量）。
*   **圆角规范**：卡片圆角 `16rpx` - `24rpx`，按钮圆角 `Full (胶囊型)`。

### 2.2 页面详细设计

#### A. 首页 (Home Page)
*   **整体布局**：沉浸式头部 + 瀑布流内容。
*   **1. 顶部导航栏 (Header)**
    *   **背景**：透明渐变到白色（随滚动变化）。
    *   **元素**：
        *   **搜索条**：全宽度胶囊型搜索框，白色半透明背景，内含灰色搜索图标和“搜索商品”占位符。
        *   **消息图标**：右侧铃铛图标 (🔔)，带红色未读数小圆点。
*   **2. 核心视觉区 (Hero Section)**
    *   **Banner**：全屏宽度的轮播图，宽高比约 16:9 或 2:1。
    *   **效果**：图片底部带有黑色渐变遮罩，保证文字可读性。
    *   **缺省状态**：若无图，显示深色背景卡片，文字居中显示“臻选好物 - PREMUIM SELECTION”。
*   **3. 金刚区 (Grid Nav)**
    *   **布局**：单行 4-5 个图标排列。
    *   **样式**：图标背景为浅色圆角矩形（如浅蓝、浅橙），图标为多彩扁平化风格。
    *   **文字**：图标下方紧跟 12px 黑色文字。
*   **4. 商品瀑布流 (Product Waterfall)**
    *   **布局**：双列布局，中间留有间隙。
    *   **卡片样式**：纯白背景，圆角，轻微阴影。
    *   **卡片内容**：
        *   商品图（正方形或 3:4）。
        *   “NEW”或“HOT”角标。
        *   商品标题（最多两行）。
        *   价格行：红色/金色价格数字（大） + 灰色原价（小，删除线） + “选购”小按钮（右下角）。

#### B. 分类页 (Category Page)
*   **交互模式**：场景化导航 + 垂直流，区别于传统的左侧菜单模式。
*   **1. 顶部搜索**
    *   固定在顶部，白色背景，灰色圆角搜索框。
*   **2. 场景导航 (Scenario Nav)**
    *   **布局**：横向滚动条 (Scroll-X)。
    *   **元素**：正方形卡片 (`140rpx`)。
    *   **状态**：
        *   **默认**：灰白背景，灰色图标。
        *   **选中**：浅蓝背景 (`#EFF6FF`)，蓝色边框，图标高亮，文字加粗。
*   **3. 商品列表**
    *   **排序条**：综合 | 销量 | 价格 (带上下箭头)。
    *   **列表样式**：单列或大图模式，强调商品细节。
    *   **快捷操作**：每个商品卡片右下角有一个显眼的 “+” 号圆圈按钮，点击加入购物车。

#### C. 商品详情页 (Product Detail)
*   **1. 轮播画廊**
    *   正方形大图，右下角有半透明胶囊显示页码 (e.g., "1/5")。
*   **2. 信息卡片**
    *   **价格区**：超大号价格数字，紧跟“会员价/代理价”金色标签。
    *   **代理商专区**（仅特定用户可见）：一个独立的圆角矩形区域，显示“分享赚 ¥XX”，带有“立即分享”按钮。
*   **3. 服务保障**
    *   横向排列的图标+文字：🛡️ 官方正品 · ⚡ 48小时发货 · ↺ 7天退换。
*   **4. 底部操作栏 (Bottom Bar)**
    *   **左侧**：购物车图标（带角标）。
    *   **右侧按钮组**：
        *   “加入购物车”：浅色/描边按钮。
        *   “立即购买”：深色/主色填充按钮。
        *   “采购入仓”（代理商特有）：特殊颜色的按钮。

#### D. 个人中心 (User Profile) - *重点设计*
*   **1. 头部卡片 (Header Card)**
    *   **背景**：深空灰/午夜蓝渐变 (`#0F172A`)，营造高端会员感。
    *   **排版**：左侧大头像（带光环边框），右侧昵称 + 等级标签（如“金牌代理”）。
    *   **右上角**：设置(⚙️) 和 消息(🔔) 白色图标。
*   **2. 资产仪表盘**
    *   悬浮在头部卡片之上的白色卡片。
    *   三列布局：累计佣金 | 可提现余额 | 团队人数。
    *   数字加大加粗，下方有“提现”或“查看”小箭头。
*   **3. 代理商工作台入口**
    *   **样式**：通栏卡片，带有装饰性背景纹理。
    *   **文字**：“代理商工作台 - 查看收益/管理团队”。
    *   **按钮**：右侧有一个“进入 >” 按钮。
*   **4. 功能列表**
    *   常规的列表项，左侧图标，右侧箭头。包含：地址管理、邀请码（支持一键复制）、关于我们等。

#### E. 弹窗与交互 (Popups & Interactions)
*   **加载中**：屏幕中央简单的旋转圆圈。
*   **下单成功**：
    *   使用 SweetAlert 风格。
    *   中央弹窗，上方显示绿色大对勾动画 ✅。
    *   文字：“支付成功”。
    *   按钮：“查看订单” | “返回首页”。
*   **规格选择**：
    *   底部半屏弹窗 (Bottom Sheet)。
    *   商品缩略图 + 价格在顶部。
    *   规格选项为圆角标签，选中态为实心色，未选中为描边。

## 3. 后台管理系统 (Admin Dashboard)

### 3.1 布局结构
*   **侧边栏 (Sidebar)**
    *   **背景**：深墨绿/深蓝灰 (`#304156`)。
    *   **Logo**：顶部固定高度，白色文字“臻选管理后台”。
    *   **菜单**：白色文字，选中项背景为蓝色 (`#409EFF`)。
    *   **层级**：支持二级折叠菜单。
*   **顶部栏 (Top Header)**
    *   白色背景，只有简单的面包屑导航和右侧用户头像/退出按钮。

### 3.2 关键页面
*   **1. 仪表盘 (Dashboard)**
    *   四张数据卡片（总销售额、订单量、用户数、利润）。
    *   下方为折线图表。
*   **2. 商品发布 (Product Edit)**
    *   **多图上传**：支持拖拽排序的图片墙。
    *   **富文本**：简洁的文本编辑器。
    *   **规格表格**：动态生成的 SKU 表格，支持批量设置价格。
*   **3. 系统设置 (Settings)**
    *   **Tab 切换**：订单配置 | 提现配置 | 佣金配置。
    *   **表单样式**：标签右对齐，输入框宽度适中，带有灰色小字提示说明。

## 4. 图标需求清单 (Icon List)
请确保以下图标风格统一（建议线性或面性二选一）：
1.  **底部导航**：首页(Home)、分类(Category)、购物车(Cart)、我的(User)。
2.  **功能图标**：搜索(Search)、消息(Bell)、设置(Gear)、扫码(Scan)。
3.  **服务图标**：正品盾牌、物流卡车、退换循环箭头。
4.  **分类图标**：根据实际商品类目设计（如美妆、食品、数码等）。
